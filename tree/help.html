LabMapManager Help

       Locations contain boxes and other locations. The tree on the left frame shows the hierarchy of locations. Clicking on a location makes it the current location. The right frame lists the boxes inside the current location.

Add New Location
Organize your boxes by creating an hierarchy of locations and sub-locations. New locations are created inside the current location. Besides the name you can enter some comments that help explain the contents of the location. On the left frame, only the names of locations are displayed. On the right frame, both the name and the comments are displayed.

Edit Location
Edit the name and the comments of the current location.

Move Location
Move the current location and its contents (boxes and sub-locations) to another parent location. For example: you might want to move your "Bookmarks/Movies/Online Tickets" to "Bookmarks/Shopping/Online Tickets".

Delete Location
Remove current location and all boxes inside. "Bookmarks" location cannot be removed.

Add New Box
Create a new box by simply entering the URL and the name of the site. Optionally add some comments to be displayed alongside with the site name.

Edit Box
Select one box in the current location by checking the checkbox on the right hand side of the listing. Then make changes to the web address, the designation, or the notes of the selected box.

Move Boxes To Another Location
Move the selected boxes from the current location to another location somewhere in the hierachies.

Delete Boxes
Remove all the selected boxes.
